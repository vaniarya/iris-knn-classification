# Iris Classification using KNN

This project implements a K-Nearest Neighbors (KNN) classifier on the Iris dataset.

## Steps covered
- Exploratory Data Analysis (EDA)
- Data visualization (scatter plots, boxplots)
- Train-test split
- KNN model training
- Model evaluation using accuracy and confusion matrix

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Result
Achieved ~93% accuracy on test data.
